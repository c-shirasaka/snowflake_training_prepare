## 1. アカウント払い出し申請
(作業時間目安: 0.5h)

Snowflake研修を受けるために必要となるサービスは以下の通り。

- Snowflake
- AWS
- Github
- HCP Terraform

以降の手順に従い、アカウント申請を行う。

### 1-1. アカウント登録
GitHubおよび HCP Terraformのアカウント登録を行う。

- **Github**: https://github.com/login
- **HCP Terraform**: https://app.terraform.io/login

### 1-2. アカウント申請
Teamsの「**Snowflakeタスク**」チャネルにて、研修アカウントの払い出し申請を行う。その際、HCP TerraformとGitHubについて、以下のアカウント情報を添付すること。  

- **Github**: ユーザー名
- **HCP Terraform**: メールアドレス

### 1-3. 確認
管理者からアカウント払い出し通知を受け取る。
当該アカウントで各サービスにログインできることを確認する。
